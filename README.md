# Hostel Expense Splitter

This is my first Python project. It helps hostel students split shared expenses fairly.

## Features
- Add expenses
- Add categories
- Track date
- Calculate total expense
- Calculate each person's share
- Show who should pay or receive money
- Save report in a text file

## Technologies Used
- Python
- datetime module
- File handling

## How to Run
1. Download the project
2. Open terminal in the project folder
3. Run:

python main.py
